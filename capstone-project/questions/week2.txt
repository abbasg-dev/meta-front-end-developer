1- With HTML 5, you should use the <div> element instead of semantic tags.
    False. In line with the proper HTML 5 semantics, you should focus instead on using the many semantic HTML 5 tags that are available today, rather than relying on the <div> element.
2- Which of the following are modern ways to build CSS layouts?
    Select all that apply.
    Grid layout. Modern CSS layouts use CSS Grid, in addition to another modern alternative.
    Flexbox layout. Modern CSS layouts use CSS Flexbox, in addition to another modern alternative.
3- True or false: Modern React components are class-based components.
    False. In modern React, components are based on functions, which was made possible with the introduction of hooks. Effectively, hooks made class-based components obsolete.
